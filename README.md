# WJXEditCode
    
## 介绍
* 批量添加代码,降低重复率!支持OC/Swift

## 版本
* 1.1.1
* 不建议使用cocopods,建议使用shell执行脚本)

    

## 准备工作
### 修改脚本
*  在WJXEditCode文件夹下找到editCode.sh文件打开修改参数
    工程名
    project_name=xxxxxx
*  混淆代码文件路径(主要是这个,这里面是你要混淆的代码所在文件夹,如果不修改则会混淆全部代码.m的)
    folder_path=${project_path}/${project_name}

###  修改示例代码
* place0.text文件
* place1.text文件
* ......
    
## 使用方法
* 使用终端进入WJXEditCode文件件
* 执行:sh editCode.sh
