# WJXEditCode
    
## 介绍
* 批量添加代码,降低重复率!支持OC/Swift

## 版本
* 1.1.1
    

## 准备工作
### 修改脚本
*  新建文件夹
*  放入editcode.sh和place文件夹(包含里面的place.m...文件)
*  拖入要添加代码的项目
*  修改项目参数folder_path是要添加代码的文件夹路径
   folder_path=${project_path}/${project_name}/xxxx


###  修改示例代码
* place0.m文件
* place1.m文件
* ......
    
## 使用方法
* 使用终端进入WJXEditCode文件
* 执行:sh editCode.sh
